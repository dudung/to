### validation
Minimal content of a SVG file is as follow

```svg
<?xml version="1.0" encoding="ISO-8859-1" standalone="no"?>
<svg version="1.1"
 width="300" height="200"
 xmlns="http://www.w3.org/2000/svg">

</svg>
```
